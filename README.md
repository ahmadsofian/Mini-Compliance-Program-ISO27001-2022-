# ISO 27001 Compliance Program – Apex Manufacturing (IT & OT Security)
This project simulates the development of a mini ISO 27001 compliance program for a fictional company, Apex Manufacturing, which produces Industrial IoT sensors for smart home environments.

The objective is to demonstrate how Governance, Risk, and Compliance (GRC) analysts design and assess a security framework across both IT and Operational Technology (OT) environments.

The project includes scope definition, asset identification, control mapping, gap analysis, and remediation planning aligned with ISO 27001 Annex A controls.
